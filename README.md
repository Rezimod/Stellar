# STELLAR — Verified Sky Observations

Observe the night sky. Capture through your telescope. Verify with satellite weather data. Collect proof on Solana.

Built by [Astroman](https://astroman.ge) — Georgia's first astronomy store.

## Stack
- Next.js 16 + React 19
- Tailwind CSS 4
- Solana Web3.js + Phantom Wallet
- FarmHawk satellite weather verification
- TypeScript

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Deployed on Vercel: [proof-of-observation.vercel.app](https://proof-of-observation.vercel.app)
