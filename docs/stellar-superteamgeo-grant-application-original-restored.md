Step 2 — Details
Project Details — What is the problem you're trying to solve, and how you're going to solve it?
PROBLEM
Georgia has the densest concentration of dark-sky sites in the South Caucasus — Kakheti, Svaneti, Tusheti — and Astroman has built a 60K-strong Georgian-speaking astronomy community over the past decade as the country's only astronomy retailer. Yet there is no Georgian-language astronomy software on the market. Telescope owners here rely on English-only apps, have no localized education content, and are entirely absent from the Solana ecosystem despite being a natural early-adopter audience for consumer crypto: passionate hobbyists, willing to spend, comfortable with new tech.

SOLUTION
Stellar is a Solana-native consumer astronomy app shipped in Georgian and English. It answers the one question every telescope owner asks — "can I observe tonight?" — with a 7-day sky forecast, planet tracker, and AI companion (ASTRA). On top of that utility layer it adds an invisible Solana economy: Privy embedded wallets created on email signup (no seed phrases), gasless transactions via a server fee-payer, compressed-NFT Discovery Attestations minted via Bubblegum for verified observations, an SPL Stars token for gamification, and Solana Pay rails for the Astroman marketplace. The thesis is utility first, crypto second — users sign up like a Web2 app, earn cNFTs for real observations verified by Claude Sonnet vision + EXIF + dedup, and only learn they're on-chain when we choose to show them.

WHY SOLANA
Compressed NFTs via Bubblegum bring per-mint cost from ~$0.50 on EVM L2s to ~$0.0001 — the only economics that make Discovery Attestations viable in a consumer flow. Gasless txs via a server fee-payer plus Privy embedded wallets make email signup feel like Web2. Solana Pay enables fiat-to-stable settlement for the Astroman marketplace. None of this is possible at these economics on any EVM chain.

WHAT THIS GRANT FUNDS
This grant funds the public-good components of Stellar — the parts the commercial market will not fund on its own. Specifically: full Georgian localization of the educational surfaces, open-sourcing the Stellar Field offline companion (Tether QVAC integration) under MIT, publishing a CC-BY Georgian-language astronomy curriculum, and a 60-day measurable on-chain user-onboarding push reported weekly to SuperteamGEO. The Astroman marketplace and prediction-markets surfaces are revenue-funded and explicitly out of scope.

Total: 5,000 USDG across 4 milestones. Each milestone closes with a public, verifiable artifact (commit, repo, PDF, or dashboard) so review is mechanical.
Milestone 1 — Georgian localization of educational surfaces
Amount: 1,500 USDG    Due: 2026-06-10
Description
Ship 100% Georgian (ka) localization across the educational surfaces of Stellar — /sky, /missions, /learn, and /observe — using next-intl. Milestone 2 — Open-source Stellar Field (Tether QVAC companion)
Amount: 1,500 USDG    Due: 2026-07-01
Description
Extract apps/field into a standalone MIT-licensed public repository with a bilingual README (EN + KA), build instructions, Android APK release, and documentation for the on-device model bundle (Llama 3.2 1B + Whisper via QVAC). Deliverable: public GitHub repo URL, signed APK release on GitHub Releases, and a "How to use Stellar Field at a dark-sky site" guide in both languages. This is a public good for any Georgian astronomer observing in remote areas without internet.

Milestone 3 — Stellar Sky Education Pack (Georgian)
Amount: 1,000 USDG    Due: 2026-07-15
Description
Author and publish 10 Georgian-language astronomy lessons — constellations, planets, lunar phases, telescope basics, observation techniques — under a CC-BY license. Deliverables: lessons live and free at /learn, downloadable PDF compilation hosted on the site, and a public announcement to the Astroman community (60K Georgian Facebook followers). First openly-licensed Georgian astronomy curriculum.
