## Step 2 - Details

### Project Details - Problem, Solution, and Execution

### 1) Problem Statement

Georgia has a growing astronomy hobbyist community but no Georgian-language astronomy software. Users rely on English-only tools for planning and learning, which creates a high barrier for first-time telescope owners. At the same time, these users are largely absent from Solana because onboarding to crypto products is usually technical and intimidating.

This leaves two gaps:

1. **Local language utility gap:** no Georgian-first astronomy education and observation workflow.
2. **Consumer onboarding gap:** no practical, utility-led path to bring non-crypto hobbyists into Solana.

### 2) Solution

Stellar is a consumer astronomy app that answers the practical question "Can I observe tonight?" with:

- 7-day sky forecast and planet visibility support
- AI assistant for telescope and observing guidance
- Observation flows that generate verifiable activity records

The product uses a utility-first design: users start with familiar web/app behavior, while on-chain actions happen in the background.

### 3) Why Solana

Solana is core to feasibility and UX:

- **Low transaction costs** make user-level actions economically viable.
- **Fast confirmations** support real-time, consumer-friendly interactions.
- **Compressed NFTs and efficient account model** enable low-cost observation attestations.
- **Gasless user flow + embedded wallets** enables email-style onboarding for non-crypto users.

This project is explicitly designed to demonstrate how Solana can onboard mainstream hobby users through practical utility instead of speculation.

### 4) Existing Solutions and Differentiation

Existing astronomy tools are mostly:

- global apps without Georgian localization, or
- utility apps with no on-chain onboarding design.

Existing crypto apps are mostly:

- crypto-first, not hobby-first, and
- not designed for astronomy-specific user journeys.

**Stellar differentiation:** Georgian-localized astronomy utility plus invisible Solana onboarding in one flow, distributed through Astroman's existing audience and retail channel.

### 5) Scope for This Grant (Realistic for up to $10k)

This request is scoped as a focused execution phase, not a full startup raise.

**In scope for grant window:**

- Georgian localization completion for core educational/observation surfaces
- Open-source packaging of Stellar Field companion app documentation and release assets
- Georgian astronomy education pack publication under an open license
- Measurable user onboarding campaign with transparent weekly reporting

**Out of scope for this grant:**

- Full product commercialization roadmap
- Large-scale paid growth
- Long-horizon features requiring larger grants

### 6) Public Goods and Ecosystem Value

This grant creates reusable ecosystem assets, not only private company value:

- Open educational content in Georgian (CC-BY)
- Open implementation and release documentation for local-first astronomy usage
- Public onboarding metrics and conversion findings for other Solana builders
