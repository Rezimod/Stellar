# Stellarr X Agent — Setup

Daily agent that drafts tweets, pings you on Telegram with one-tap Approve / Reject, and posts approved drafts to **@stellarrclub** on X.

- **2 posts per day** via Vercel cron (`vercel.json`):
  - **12:01 Tbilisi** (08:01 UTC) — midday: development updates, product features, build notes
  - **18:00 Tbilisi** (14:00 UTC) — evening: Solana infrastructure, astronomy utility, space context
- Post types: `short_post`, `sky_verdict`, `product_spotlight`, `astro_fact`, `space_news`, `build_update`, `solana_infra`
- Copy is generated by `gpt-4o-mini` with a Stellarr brand-voice system prompt (no emojis, no hype, X hashtags included).
- Images are generated with OpenAI, then composited with the supplied Stellar logo, readable post text, and a Solana badge for `solana_infra`.
- All drafts stored in Neon (`tweet_drafts` table) for audit.

## 1. Get X API access (10 min)

**X Premium ≠ X API credits.** Premium is for the consumer app; posting via this agent uses the [Developer API](https://developer.x.com), which needs its own credits. If credits are empty, Approve opens **X compose** with the draft text (free) so you can still post manually.

X free tier: 500 posts/month — plenty for one a day (when credits are loaded).

1. Go to https://developer.x.com → sign in with the **@stellarrclub** account.
2. Apply for a **Free** developer account (instant approval for most accounts).
3. Inside the developer portal, create a **Project**, then inside it create an **App**.
4. App settings → **User authentication settings** → Set up:
   - App permissions: **Read and write**
   - Type of app: Web App (callback URL can be `https://stellarr.club`)
   - Save.
5. **Keys and tokens** tab → generate / regenerate all four:
   - `API Key` → `X_API_KEY`
   - `API Key Secret` → `X_API_SECRET`
   - `Access Token` (must show "Read and write") → `X_ACCESS_TOKEN`
   - `Access Token Secret` → `X_ACCESS_TOKEN_SECRET`

   **Important:** if your Access Token shows "Read only", regenerate it AFTER setting permissions to Read+Write in step 4. Read-only tokens silently fail at `tweet()` time.

## 2. Get a Telegram approval bot (3 min)

1. Open Telegram → search `@BotFather` → `/newbot` → name it `Stellarr X Agent`.
2. Copy the token → `TELEGRAM_BOT_TOKEN`.
3. Find your bot in Telegram, send it `/start` (any message).
4. In a browser, open `https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates` → find `"chat":{"id":1234567890,...}` → that number is `TELEGRAM_CHAT_ID`.

## 3. Generate secrets (10 sec)

```bash
openssl rand -hex 32   # → X_AGENT_SECRET
openssl rand -hex 32   # → CRON_SECRET
```

## 4. Paste env vars

Add the eight new vars from `.env.example` (X_*, X_AGENT_SECRET, CRON_SECRET, TELEGRAM_*) to:
- `.env.local` (for local testing)
- Vercel project → Settings → Environment Variables (for production)

`OPENAI_API_KEY` is already set — the agent reuses it.

## 5. Push schema + deploy

```bash
npm run db:push   # creates the tweet_drafts table on Neon
git add -A && git commit -m "feat(agent): daily X poster" && git push
```

Vercel auto-deploys. The cron registers on next deploy.

## 6. Test it once manually

After deploy, hit the route by hand (with your `CRON_SECRET`):

```bash
curl -H "Authorization: Bearer $CRON_SECRET" \
  https://stellarr.club/api/agent/draft-tweet?skipImage=1
```

You should:
1. See a JSON response with the draft + approve/reject URLs.
2. Get a Telegram message from your bot with the same links.
3. Tap **Approve & post** → a confirmation page opens, and the tweet goes live on @stellarrclub.

If the post fails, the page shows the X API error and the draft is marked `failed` in Neon (so cron won't retry the same one).

## 7. Done

From now on, you'll get **2 Telegram pings per day** (12:01 and 18:00 Tbilisi). Approve each when ready.

## Tuning

- **Change post times:** edit `vercel.json` cron schedules (UTC).
- **Force a slot manually:** `?slot=midday|evening` on the draft URL.
- **Force a post type:** `?kind=short_post | sky_verdict | space_news | product_spotlight | astro_fact | build_update | solana_infra`.
- **Edit the brand voice:** `BRAND_SYSTEM` in `src/lib/tweet-agent.ts`.
- **Add/remove product spotlights:** `PRODUCT_SPOTLIGHTS` in `src/lib/tweet-agent.ts`.
- **Add/remove Messier targets:** `MESSIER_TARGETS` in same file.

## Cost

- X API free tier: 500 posts/month — covers 2/day (~60/month) when credits are loaded.
- OpenAI: ~$0.001 per draft — negligible.
- Vercel cron: included on Hobby.
- Spaceflight News + Open-Meteo: free.

Total: pennies/month.

## Killing it

To pause: delete the cron entry from `vercel.json` and redeploy. Or comment out the `crons` block. Drafts already in Neon are kept.
